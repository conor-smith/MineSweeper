# Minesweeper

A project with the following goals
- Refresh my c knowledge
- Learn more about make
- Learn ncurses
- Learn some graphical libraries

## Dependencies
- [SDL2](https://www.libsdl.org/) (Not currently compatible with SDL3)
- [ncurses](https://invisible-island.net/ncurses/announce.html)

## Build
To build GUI, simply run
```
make
```

To build only the textui, run
```
make textui
```
